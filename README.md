#### 收集歌词
我用之前文章爬取歌词的方法，爬取了中国一些摇滚歌手的歌词，选取了以下摇滚歌手的歌词：崔健、许巍、郑钧、汪峰、轮回乐队、黑豹乐队、唐朝乐队、窦唯、 高旗&超载乐队、指南针乐队。每位歌手 50 首歌左右，总结收集了 500 首歌的歌词，歌词字数有 12 万多。
